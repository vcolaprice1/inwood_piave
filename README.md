# Atlante idrografico del Piave — 1891

WebGIS storico degli opifici e dei corsi d'acqua del bacino del Piave, ricavato dalla *Carta idrografica d'Italia* del 1891.

La mappa consente di esplorare gli opifici per tipologia e consultarne le schede, cercare per denominazione, leggere i nomi dei corsi d'acqua e attivare sia il contesto amministrativo storico sia le scansioni della cartografia originale.

Il sito è interamente statico e può essere pubblicato direttamente con GitHub Pages.
